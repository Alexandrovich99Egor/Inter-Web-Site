#Здравстуйте!
Для того что бы запустить проэкт,необходимо :
1.Установить npm пакет (модули зависимости) - в терминале пишем : npm install или npm i 
ВНИМАНИЕ прописывать в директории где есть файл packedge.json.
2.Далее после установки нод модулей можем приступать к запуску проекта для этого заходим в папку front 
(в ней лежит наш проєкт)и пишет в терминале npm start.
Советы,лично я бы сделал отпраку данных на сервер на php $_POST запросом,специльально для этого создал бабку back - 
тут будет Ваш бекенд лежать.

Немного о проэкте,написан он на библиотеке JS - React.js,используется препроцессор SCSS (CSS),все данные 
которые касаются контента находятся в папочке data там обьекты с данными.

ВАЖНО!проверьте какой стоит у вас Node.js
